# Law Office Website

وب‌سایت و پنل مدیریت اختصاصی برای موسسه حقوقی، با تمرکز روی طراحی رسمی، تیره و طلایی.

## هدف پروژه

این پروژه قرار است یک سایت کامل برای موسسه حقوقی باشد که شامل سایت عمومی و پنل مدیریت محتوا است.

## بخش‌های اصلی

- صفحه اصلی
- معرفی موسسه
- درباره ما
- ارتباط با ما
- اخبار و مقالات
- وبلاگ
- پنل مدیریت

## تکنولوژی‌ها

- Next.js
- TypeScript
- MongoDB
- Tailwind CSS

## مسیر اجرا

1. راه‌اندازی اولیه پروژه
2. پیاده‌سازی رابط کاربری سایت
3. پیاده‌سازی رابط کاربری پنل مدیریت
4. اتصال به دیتابیس
5. ساخت API و CRUD
6. احراز هویت مدیران
7. آماده‌سازی برای انتشار
